# Coca-Cola Maiduguri Enterprise Security E-Register

A comprehensive security management system for Coca-Cola operations in Maiduguri, Nigeria. This application provides real-time monitoring, incident tracking, and compliance reporting for enterprise security operations.

## Features

- **Security Incident Tracking**: Log and manage security incidents
- **Personnel Management**: Track security personnel assignments
- **Patrol Scheduling**: Manage patrol routes and schedules
- **Real-time Alerts**: Instant notifications for security threats
- **Compliance Reporting**: Generate security compliance reports
- **Access Control**: Monitor and control facility access
- **Incident Analytics**: View security statistics and trends

## Project Structure

- `index.html` - Main security dashboard
- `styles.css` - Application styling
- `script.js` - Core functionality
- `README.md` - This file

## Getting Started

1. Open `index.html` in a web browser
2. The application will load with sample security data
3. Use the navigation menu to access different security modules

## Usage

### Logging Incidents
1. Navigate to "Incidents" section
2. Click "New Incident"
3. Fill in incident details and submit

### Managing Personnel
1. Go to "Personnel" section
2. View assigned security staff
3. Update assignments as needed

### Viewing Reports
1. Access "Reports" section
2. Select date range and incident type
3. Generate and download reports

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
